### About

I work at the intersection of **quantum computing** and **machine learning** — implementing
algorithms from the math up rather than from a library down. My repositories span quantum
singular value transformation, variational algorithms (VQE/QAOA), quantum kernels and QCNNs,
and physics-informed / neuro-symbolic models. I also build and ship full-stack applications
(Laravel, TypeScript/React) end to end.

- 🔬  Quantum: **QSVT, VQE, QAOA, quantum kernels, gate decomposition, Qiskit tooling**
- 🧠  ML/AI: **graph neural networks, neuro-symbolic learning, RAG systems**
- 🛠️  Web: **Laravel/PHP and TypeScript/React** APIs and interfaces
- 🌍  Based in Egypt · working in English &amp; Arabic
